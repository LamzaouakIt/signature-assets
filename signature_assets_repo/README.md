# signature-assets

Assets pour les signatures email Gmail (images hébergées, jamais en pièce jointe).

## Contenu

| Fichier | Rôle |
|---|---|
| `signature.html` | Page de la signature — ouvrir dans un navigateur, copier, coller dans Gmail |
| `phone.png` / `mail.png` / `web.png` / `pin.png` | Icônes 28x28 (charte #e8672b), affichées en 14x14 |
| `said.jpg` | Photo de profil 140x140 (affichée en 70x70) |

## URLs des assets

```
https://raw.githubusercontent.com/LamzaouakIt/signature-assets/main/<fichier>
```

## Installation dans Gmail

1. Ouvrir `signature.html` dans un navigateur (vérifier que toutes les images s'affichent)
2. Sélectionner la signature → Ctrl+C
3. Gmail → ⚙️ Paramètres → Voir tous les paramètres → Signature → Créer → Ctrl+V → Enregistrer

## Ajouter un nouvel employé

1. Uploader sa photo `prenom.jpg` (carrée 140x140, < 50 Ko)
2. Dupliquer `signature.html`, adapter nom / poste / tél / photo
